Step one - description
This example show Dynamic Step Process Panels with content loaded from files.

Features:
- content of first and second step loaded via ajax from files,
- 3 steps,
- step-by-step mode,
- every step can be choosed again,
- fade effect speed 1 second,
- first step selected,
- style green-orange,
- previous and next buttons,
- custom previous button caption,
- custom next button caption,
- step labels
